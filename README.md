# Telegram Ticket Checker Bot

Telegram-бот на Python с использованием библиотеки Aiogram и Selenium. Он мониторит сайт с расписанием билетов, проверяет их наличие и рассылает уведомления подписчикам в Telegram. Также бот обрабатывает команды пользователей (/start, /info) и отвечает по запросам.

Для запуска: ```uv run TicketAlertBot ```
